# Hotel Madness: Greek Management RPG

A high-stakes, AI-powered text RPG simulating the intense, toxic, and high-pressure environment of a modern Greek hotel.

Powered by Google Gemini as the Game State Engine (Dungeon Master).

## Setup Instructions

1. **Install Dependencies**
   Open a terminal in this directory and run:
   ```bash
   npm install
   ```

2. **Configure API Key**
   - Copy `.env.example` to a new file named `.env.local`
   - Get a free API key from [Google AI Studio](https://aistudio.google.com/)
   - Paste your key into `.env.local`: `VITE_GEMINI_API_KEY=your_key_here`

3. **Run the Game**
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` in your browser.

## Features
- **3 Dynamic Career Paths:** Receptionist, Waiter, and Chef.
- **AI Dungeon Master:** Every scenario is dynamically generated by Gemini according to strict corporate lore.
- **Premium UI:** A sleek, dark-mode corporate dashboard to track your Stress, Reputation, Cash, and hidden metrics.
- **Deep Mechanics:** Manage staff relations to avoid informants, survive GM micro-management, and navigate Black Swan events like grid failures.
